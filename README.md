# 🍣 Pepo's Sushi Restaurant

Projeto desenvolvido como parte da disciplina de **Projeto Integrador** do **Instituto Federal Catarinense – Campus Araquari**.

O objetivo do projeto foi aplicar conhecimentos de desenvolvimento web e figma por meio da criação de uma página institucional fictícia, escolhida por nós sobre culinária japonesa, priorizando design moderno, responsividade e experiência do usuário.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Google Fonts
